# Reference Dashboards

Reference screenshots / exports of existing logistics dashboards, kept for
design and feature reference while building the Logistics Control Tower /
Value Cockpit.

## Contents

`Reference_Dashboards.zip` contains:

- B2B Operations Dashboard — Overview (PDF), Overview Detailed View, Message Details
- Brokerage Dashboard — Overview, Details, Details Table, Exception Queue, Exception Queue Table, Traffic Report
- Managed Trans Dashboard — Overview, Details, Details Table
- Value Cockpit — Home Screen

These are reference material only; they are not part of the application build.
